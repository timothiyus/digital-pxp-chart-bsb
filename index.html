GameChanger Import

Season, conference, and box-score CSVs

For GCHS and other GameChanger-sourced rosters. Imports full-season and conference CSVs into the active side's roster. Existing players (by side + number + name) get updated; new ones are added.

PrestoSports Import

NJCAA / GCCC roster + per-slice CSVs

For GCCC and other PrestoSports-sourced rosters. Drop the roster .TRX or Safari-downloaded .TRX.txt first, then the four overall-season stat CSVs and (optionally) the four conference-only CSVs. Roster names can be First Last or Last, First. Each stat slot fills its own slice of player.stats (overall) or player.confStats (conference). The matching .TRO file is index-only.

Box Score Import

One file per game; per-player line

A box score CSV or TXT captures one game's line for each player who appeared. Imported box scores show up as a Recent Games strip in the Active Batter Detail and on each player's roster card. CSV headers should include Number, Last, First plus stat columns (AB, H, 2B, 3B, HR, BB, SO, RBI, ...). TXT imports are designed for GameChanger/MaxPreps table exports.

Clear cached imports

Wipe imported documents and their stats

Removes all GameChanger and PrestoSports source records, every imported box score, and resets each player's season + conference stat lines back to zero. Roster identity (number, name, position) and live scorecard data are preserved. Use this when you want to re-import fresh CSVs without stale data merging in.

Notation cheat sheet

Type in the diamond center

Type any of these tokens in the freetext field at the center of a diamond. Stat-bearing tokens (bold) auto-update the batter's stat line and the inning totals. Anything else is recorded as flavor text and shown on the chart with a gold tint.

Storage

Everything saves to your browser's localStorage under pxp-baseball-chart-v1. Use Reset (top right) to wipe and start over. Export and import workspace JSON from the Export tab when moving a full chart between devices.
